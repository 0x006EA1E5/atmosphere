62  /**
63   * A Broadcaster is responsible for delivering messages to its subscribed
64   * {@link AtmosphereResource}, which are representing a suspended response.
65   * {@link AtmosphereResource} can be added using {@link Broadcaster#addAtmosphereResource},
66   * so when {@link #broadcast(java.lang.Object)} execute,
67   * {@link AtmosphereHandler#onStateChange(org.atmosphere.cpr.AtmosphereResourceEvent)} will
68   * be invoked and the suspended connection will have a chance to write the
69   * message available using {@link AtmosphereResourceEvent#getMessage()}
70   * <br>
71   * A {@link Broadcaster}, by default, will use an {@link ExecutorService}, and the
72   * number of Thread will be computed based on the core/cpu of the OS under
73   * which the application run. Thus invoking {@link org.atmosphere.cpr.Broadcaster#broadcast(Object)} will be executed
74   * <strong>asynchronously</strong> so this is important to wait for the {@link Future#get} to awake/unblock to be garantee
75   * the  operation has completed.
76   * <br>
77   * One final word on Broadcaster: by default, a Broadcaster will broadcast using
78   * all {@link AtmosphereResource} on which the response has been suspended, e.g. {AtmosphereResource#suspend()}
79   * has been invoked. This behavior is configurable and you can configure it by invoking the
80   * {@link Broadcaster#setScope(org.atmosphere.cpr.Broadcaster.SCOPE)} ):<ul>
81   * <li>REQUEST: broadcast events only to the AtmosphereResourceEvent associated with the current request.</li>
82   * <li>APPLICATION: broadcast events to all AtmosphereResourceEvent created for the current web application.</li>
83   * <li>VM: broadcast events to all AtmosphereResourceEvent created inside the current virtual machine.</li>
84   * </ul>
85   *
86   * @author Jeanfrancois Arcand
87   */

98      /**
99       * Set the maximum number of suspended {@link AtmosphereResource}. If the max is reached, Atmosphere will either
100      * resume {@link AtmosphereResource} using {@link org.atmosphere.cpr.Broadcaster.POLICY#FIFO} (first in first out)
101      * or {@link org.atmosphere.cpr.Broadcaster.POLICY#REJECT} the {@link AtmosphereResource}.
102      * <p/>
103      * By default the number is uunlimited.
104      *
105      * @param maxSuspended max suspended
106      * @oaram policy the {@link org.atmosphere.cpr.Broadcaster.POLICY}
107      */
108     void setSuspendPolicy(long maxSuspended, POLICY policy);

110     /**

117     <T> Future<T> broadcast(T o);

153     /**

157      *
158      * @param o       and {@link Object} to be broadcasted.

160      * @param period  The period inbetween broadcast.

162      * @return a {@link Future} that can be used to synchronize using the {@link Future#get()}
163      */
