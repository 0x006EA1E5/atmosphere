The Atmosphere Framework

Real Time Client Server Framework for the JVM, supporting WebSockets and Cross-Browser Fallbacks Support

What is The Atmosphere Framework?

The Atmosphere Framework is the most popular asynchronous application development framework for enterprise Java. The Atmosphere Framework provides the enterprise features required to build massive scalability real time asynchronous application

Write Once, Deploy Anywhere

The Atmosphere Framework goal to make real time apps possible in every browser, library and mobile device, blurring the differences between the different transport mechanisms. The framework is written in Java and Javascript and support the majority of existing Java/Web Framework and can be deployed in every Java Web Server like Tomcat, Jetty, GlassFish, Netty etc.

Client

$(function () {
   var request = {
     url: document.location.toString() + 'echo',
     transport : "websocket" ,
     fallbackTransport: 'long-polling'};

   request.onMessage = function (response) {
     console.log(response.responseBody)
   };
   $.atmosphere.subscribe(request).push("Hello");
}
