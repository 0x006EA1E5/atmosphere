<h1>Welcome to Atmosphere, a portable <a
        href="http://jfarcand.wordpress.com/2007/05/15/new-adventures-in-comet-polling-long-polling-or-http-streaming-with-ajax-which-one-to-choose/">AjaxPush/Comet</a>
    and WebSocket Framework</h1>

<p>Atmosphere is a POJO based framework using
    <a href="http://en.wikipedia.org/wiki/Inversion_of_Control">Inversion of Control (IoC)</a>
    to bring push/<a href="http://jfarcand.wordpress.com/20.6.3/07/06/the-grizzly-comet-or-why-space-shuttle-discovery-launch-was-delayed/">Comet</a>
    and Websocket to the masses! Finally a framework which can run on any Java based Web Server, including
    <a href="http://tomcat.apache.org">Tomcat</a>, <a
            href="http://www.mortbay.org/jetty-6/">Jetty</a>,
    <a href="http://glassfish.dev.java.net">GlassFish</a>, <a
            href="http://www.oracle.com/appserver/weblogic/weblogic-suite.html">Weblogic</a>,
    <a href="http://grizzly.dev.java.net/">Grizzly</a>, <a href="http://www.jboss.org/jbossweb/">JBossWeb and JBoss</a>,
    <a href="http://www.caucho.com/">Resin</a>, etc.
    without having to learn how Comet or WebSocket support
    has been differently implemented by all those Containers. The Atmosphere Framework has both client (JQuery PlugIn) and server component.</p>

<p>Servlet 3.0 is supported along with framework like Jersey, GWT, Struts, Spring, Wicket etc. and programming language
    like JRuby, Groovy and Scala. Finally, any web application can be clustered using our Cluster plugin architecture
    supporting JMS or JGroups.</p>

<h2 class="sidebar-title">Getting started using Maven</a></h2>
If you are using Maven, just add the following dependency:<pre><code>    &lt;dependency&gt;
        &lt;groupId>org.atmosphere&lt;/groupId&gt;
        &lt;artifactId>atmosphere-{atmosphere-module]&lt;/artifactId&gt;
        &lt;version>0.6.3&lt;/version&gt;
    &lt;/dependency&gt;</code></pre>

Where atmosphere-module can be: jersey, runtime, guice, bayeux, cluster or spade-server. Our official version are released in the Maven central repository. You can download the artifacts
directly from <a href="http://oss.sonatype.org/content/repositories/releases/org/atmosphere/">here</a>

<p>Atmosphere 0.6.3 is our <a href="http://jfarcand.wordpress.com/2010/06/24/atmosphere-0.6.3-released-transparent-websocket-and-comet-support-jquery-support-more-adoption/">official release</a>, and our work in progress version is 0.7,
    targeted for End August 2010.</p>

<p>We are using Maven 2 and Github. To checkout Atmosphere, just do <strong>git clone
    git://github.com/Atmosphere/atmosphere.git</strong>. If you are interested, <a
        href="http://java.net/projects/atmosphere/lists"/>subscribe</a> to our mailing lists
    (user@atmosphere.java.net or dev@atmosphere.java.net) for more info!</p>
<p>We are on irc.freenode.net under #atmosphere-comet</p>
<p>Browse our mailing lists: <a id="nabblelink"
